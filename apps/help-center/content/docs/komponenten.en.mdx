## CodeTabs

Tabs for API examples in cURL, JavaScript, Python and Java.

<CodeTabs
  tabs={{
    curl: `curl https://api.goodfunds.example/v1/donations \\
  -H "Authorization: Bearer $TOKEN"`,
    js: `const res = await fetch("https://api.goodfunds.example/v1/donations", {
  headers: { Authorization: \`Bearer \${token}\` },
});
const donations = await res.json();`,
    python: `import requests

res = requests.get(
    "https://api.goodfunds.example/v1/donations",
    headers={"Authorization": f"Bearer {token}"},
)
donations = res.json()`,
    java: `HttpRequest req = HttpRequest.newBuilder()
    .uri(URI.create("https://api.goodfunds.example/v1/donations"))
    .header("Authorization", "Bearer " + token)
    .build();`,
  }}
/>
