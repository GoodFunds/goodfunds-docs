## So funktioniert eine Integration

Die meisten Integrationen folgen demselben Muster:

1. **Zugangsdaten anlegen** — beim externen Anbieter (z. B. ein API-Schlüssel oder eine
   App-Registrierung).
2. **In GoodFunds eintragen** — unter **Integrationen** die Zugangsdaten hinterlegen.
3. **Verbindung testen** — GoodFunds prüft, ob die Verbindung steht.
4. **Felder zuordnen** — festlegen, welches GoodFunds-Feld in welches Feld des anderen Systems
   geschrieben wird.

<Callout type="info">
  Deine Zugangsdaten werden in GoodFunds immer **verschlüsselt** gespeichert. Geheime Schlüssel zeigen
  wir nach dem Speichern nicht mehr im Klartext an.
</Callout>
