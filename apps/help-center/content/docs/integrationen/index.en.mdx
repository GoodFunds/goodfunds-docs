## How an integration works

Most integrations follow the same pattern:

1. **Create credentials** — at the external provider (e.g. an API key or an app registration).
2. **Enter them in GoodFunds** — add the credentials under **Integrations**.
3. **Test the connection** — GoodFunds checks that the connection works.
4. **Map fields** — decide which GoodFunds field is written to which field in the other system.

<Callout type="info">
  Your credentials are always stored **encrypted** in GoodFunds. Secret keys are never shown in plain
  text again after you save them.
</Callout>
